A Cross Platform Tail Company gear control App

## Features

- Support for the same actions/moves from Crumept
- Triggers for walking, shaking and other gestures
- Custom Actions
- Joystick for moving gear
- Dark Mode

## Have a suggestion?

- Manual OTA Updates

- Access to app logs
